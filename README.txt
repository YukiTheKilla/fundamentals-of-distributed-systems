Для запуска всего проекта необходимо запустить main_starter.py
После запускаем machine/xlsx_manufacture.py
Указываем количество карт для генерации (оптимально 300000)
Указываем количество строк в файле (желательно не более 20000)
Запускаем клиент серез client/app.py
Заходим в браузер и вбиваем http://127.0.0.1:5000
Далее указываем имя и пароль, или создаем.
Выбираем файл для скачивания и нажимаем на кнопку скачать
Готовый файл сохраняется в папку client/downloads
